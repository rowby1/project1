## 개발환경
* Java JDK 1.8.0_301
* Eclipse Java Development version 2021-06-R
* Oracle Database 11g Express Edition 11.2.0.2

## 사용기술
* Java Swing
* Naver 검색 API (JSON)
* data.go.kr API (XML)
* JSON : json-simple-1.1.1.jar 외부 라이브러리
* XML : DOM Parser (Java 내장)
